# wxRadio
Project for transmitting weather data from remote location. See the current [Dashboard](https://busbykt.pythonanywhere.com/).


## Equipment
- RaspberryPi4
- RFM9X M0 900MHz Radio @ 915MHz

## Hardware Preparation
